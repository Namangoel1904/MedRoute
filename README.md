# MedRoute Emergency Allocation MVP

MedRoute is a static, dependency-free MVP for a smart emergency healthcare resource allocation system. It demonstrates the first-round hackathon loop: ambulance intake, AI-style hospital recommendation, hospital command queue, resource readiness, and impact metrics.

## Open the MVP

Open `index.html` in a browser.

No install step is required.

## Demo Flow

1. Review the live dispatch overview at the top of the app.
2. Use the ambulance intake form to change emergency type, vitals, severity, and location.
3. Watch the recommendation engine rank hospitals by distance, specialty fit, and available capacity.
4. Click `Dispatch to recommended hospital` to send the case into the hospital command queue.
5. Use `Accept and prepare` to simulate pre-arrival coordination.
6. Use `Simulate Case` to cycle through cardiac, trauma, stroke, and respiratory examples.

## MVP Coverage

- Real-time patient data entry
- AI-style best hospital recommendation
- Severity classification
- Hospital command dashboard
- Pre-arrival resource checklist
- Capacity and equipment visibility
- Impact metrics aligned with SDG 3

## Suggested Pitch Line

MedRoute reduces emergency response lag by deciding the right destination before arrival, giving hospitals time to prepare the exact resources a patient needs.
