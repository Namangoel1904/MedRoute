# MedRoute 🏥

![Dashboard Preview](assets/dashboard_preview.png)

## 📖 Overview

**MedRoute** is a high-fidelity, real-time administrative portal designed for modern hospitals to streamline emergency operations. It provides a centralized "Command Center" for monitoring critical resources, triaging incoming ambulance cases, and managing hospital-wide capacity with precision.

Designed with a focus on high-stress environments, MedRoute offers a premium, responsive interface that ensures vital information is always at the fingertips of hospital administrators and charge nurses.

## ✨ Key Features

### 🚀 Command Center (Dashboard)
- **Live Metrics**: Monitor available beds, ICU capacity, and ventilator status in real-time.
- **Critical Alerts**: Instant notification system for capacity thresholds and equipment failures.
- **Incident Overview**: Quick-view list of high-priority incoming cases.

### 🚑 Case Management & Triage
- **Priority-Based Queue**: Automated sorting of incoming cases (Critical, Urgent, Moderate).
- **Patient Detail Cards**: Deep-dive into patient history, vitals (BP, HR, SpO2), and allergies.
- **Resource Allocation**: Seamless workflow to assign units (ICU, Cath Lab, ER) and specific beds before the patient arrives.
- **Diversion Management**: Ability to decline and redirect cases to neighboring facilities with automated redirect notifications.

### 🛏️ Resource & Staffing
- **Interactive Bed Grid**: Visual representation of unit capacity with state toggling (Available, Occupied, Cleanup).
- **Equipment Tracking**: Real-time stats for ventilators, monitors, and defibrillators.
- **Shift Planner**: Visualization of coverage gaps and on-call rosters for cardiology, trauma, and other departments.

### 📊 Analytics & Reporting
- **Performance Tracking**: Door-to-bed time analytics and acceptance rate trends.
- **Occupancy Trends**: Visualized trends for ICU and General Ward occupancy.

## 🛠️ Technology Stack

- **Frontend**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Tailwind CSS (Utility-first framework)
- **Icons**: Font Awesome 6
- **Layout**: CSS Grid & Flexbox for high-density information display

## 🚀 Getting Started

### Prerequisites
- Any modern web browser (Chrome, Firefox, Edge, Safari).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/MedRouteSub.git
   ```
2. Navigate to the project directory:
   ```bash
   cd MedRouteSub
   ```
3. Open `index.html` in your browser:
   ```bash
   # On Windows
   start index.html
   
   # On macOS
   open index.html
   ```

## 🔐 Security Notice
This prototype is designed to simulate a secure environment. In a production setting, this system should be integrated with secure SSO (Single Sign-On) and HIPAA-compliant data handling protocols.
